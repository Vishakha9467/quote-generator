# Quote Generator

A web-based application that delivers personalized quotes based on user input. Users can enter a keyword to fetch quotes from a CSV file, categorized by topics like life, love, motivation, and more.

## Features
- Fetches quotes from a CSV file.
- Allows users to input a keyword for category-based filtering.
- Responsive and visually appealing interface.
- Built with Python Flask and HTML/CSS for seamless interaction.

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/quote-generator.git

2. Navigate to the project folder:
   ```bash
   cd quote-generator

3. Install dependencies:
   ```bash
   pip install -r requirements.txt

4. Run the application:
   ```bash
   python app.py

## Usage
- Open the application in your browser (http://127.0.0.1:5000).
- Enter a keyword to search for quotes (e.g., life, happiness).
- Enjoy your personalized quote with the author displayed.

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- Data Source: CSV File
